# eGPS-Base Docs

This directory contains the repository-local documentation for `egps-base`.
It is meant to help readers move from the project overview into the source tree quickly.

[中文](README_zh.md)

## Start Here

- [global_understand.md](global_understand.md) - what the repository contains and how it is organized
- [understand_packages1_core_structure.md](understand_packages1_core_structure.md) - core structures, phylogeny-related code, and genetic codes
- [understand_packages2_file_io.md](understand_packages2_file_io.md) - FASTA, TSV, and MSA input/output code
- [understand_packages3_parsers.md](understand_packages3_parsers.md) - parsers for database files such as NCBI taxonomy, BLAST, and Pfam, plus REST and data-analysis code
- [understand_packages4_utilities.md](understand_packages4_utilities.md) - utility layer, CLI tools, GUI helpers, and unified output
- [understand_packages5_visualization.md](understand_packages5_visualization.md) - graphics, tree display, and tanglegram-related code
- [design/axis_tick_calculation.md](design/axis_tick_calculation.md) - design note for automatic axis tick calculation in 2D graphics
