# 从树结构到图形绘制：我们开源了 eGPS 2.1 的基础能力库

如果你正在做生物信息学、系统发育分析，或者经常需要处理 FASTA、TSV、多序列比对、系统发育树和注释数据，那么你可能会遇到一个共同问题：很多工作并不难，但总在重复造轮子。`egps-base` 就是为了解决这类问题而开源的基础库。

`egps-base` 是 eGPS 2.1 生态中已经开源的底层 Java 基础库，定位不是“大而全的平台”，而是一组可以直接复用、自由组合的核心能力模块。它覆盖了树结构与系统发育分析、FASTA 与 MSA 读写、TSV 表格处理、BLAST/Pfam/NCBI taxonomy 等常见生信数据解析，以及图形绘制、树显示、tanglegram、REST 集成和若干命令行/小型 GUI 工具。

这意味着，开发者既可以把它当作自己的项目依赖，快速搭建数据处理流程；也可以把它当作一个清晰的源码仓库，直接查阅具体实现，理解某类生信基础问题是如何落地成代码的。尤其是树结构编解码、可视化绘制、坐标轴刻度自动计算这类细节，在仓库中都有较清楚的实现与配套文档。

这次我们也同步整理了项目文档，对仓库结构、核心包、输入输出、解析器、工具层和可视化模块做了重新梳理，让第一次接触这个项目的读者也能更快找到入口。

如果你希望基于现有能力扩展自己的分析工具，或者想找一个偏“工程实现型”的生信 Java 代码库做参考，欢迎关注和使用 `egps-base`。
